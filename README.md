Expense Tracker

This is a simple C++ project made for keeping track of daily expenses.
The program lets you add new expenses with the reason and amount, and it also records the date and time automatically.
All the data is saved in a text file so that the expenses are available even after closing the program.

Main Features
• Add new expense with reason and amount
• View all saved expenses with total spent amount
• Stores data permanently using a file (expenses.txt)
• Menu based interface for easy use

How to Run

Open the project in Dev C++ or any C++ IDE

Compile and run the main file

Choose options from the menu to add or view expenses

This project was created as a simple practice for learning C++ basics like structures, vectors, file handling and time functions.
